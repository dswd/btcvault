BTC Vault 0.3: Feature release (2013-11-23)

Changes

  • Added MultiBit client
  • Updated Electrum to version 1.9.4
  • Updated local copies of Bitaddress.org and Brainwallet.org
  • Sorted icons on desktop

Release files

  • btcvault-0.3.iso (for installation on CD-Rom, 272 MiB)
  • btcvault-0.3.zip (for installation on USB stick, 272 MiB)
  • 9999-btcvault.sb (for usage on a Slax system, 1.2 MiB)
  • 9998-electrum.sb (for usage on a Slax system, 0.6 MiB)
  • 9997-multibit.sb (for usage on a Slax system, 7.3 MiB)

BTC Vault 0.2: Bugfix & feature release (2013-04-06)

Changes

  • Added local copy of bitaddress.org with shortcut on desktop and in menu
  • Disabled flash plugin by default
  • Disabled Xorg server network functionality
  • Added 7z encryption capability

Release files

  • btcvault-0.2.iso (for installation on CD-Rom, 232 MiB)
  • btcvault-0.2.zip (for installation on USB stick, 232 MiB)
  • 9999-btcvault.sb (for usage on a Slax system, 0.5 MiB)
  • 9998-electrum.sb (for usage on a Slax system, 0.4 MiB)
  • Release checksums

BTC Vault 0.1: Initial release (2013-03-31)

Changes

  • Electrum software, connected with .wallet files, link on desktop

  • Keyboard chooser got German, French and Spanish layouts
