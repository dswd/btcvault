BTC Vault

Welcome to BTC Vault, a Linux distribution designed to securely handle bitcoin wallets.

Features

BTC Vault is based on Slax and thus contains a full Linux installation including the KDE desktop environment and the Firefox webbrowser.
Also, BTC Vault includes the Electrum bitcoin client that you can use to manage your coins.
Some additional modifications include:
  • All Electrum wallets (electrum.dat and all .wallet files) will open with Electrum.
  • You can create new wallets by using the action from the folder context menu or the desktop icon.
  • Firefox supports the bitcoin: url-scheme.
  • Links on the desktop lead to Blockchain.org and a local copy of Brainwallet.org.
  • Links on the desktop and the panel lead to the home directory.
  • Populated the keyboard chooser in the panel with German, French and Spanish layouts.

Recommendations

  • Use BTC Vault to create a safe wallet and store it on a dedicated USB stick. Print out the seed and store that in a safe location.
  • Create a read-only wallet to check your balance on your normal computer as described in the section Electrum.
  • Only start up BTC Vault and open your wallet when you want to spend some coins.
  • If you want to, create a second wallet containing only a small amount of coins (that you can risk losing) on your normal computer for small transactions.
